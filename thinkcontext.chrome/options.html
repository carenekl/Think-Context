Welcome to thinkContext, the web browser extension which adds contextually relevant, politically progressive information to your web browsing.

As you browse the web thinkContext consults several progressive data sources and if it finds anything relevant to what you are looking at it will unobtrusively add that information right into the page.

Below you may configure which data sources thinkContext consults. We recommend to begin with that you enable all data sources and only disable one if you have a specific reason.


Progressive News Trackback - When reading mainstream news thinkContext will let you know what progressive news sources are saying about what you are reading. thinkContext scans over 50 progressive news sources such as The Nation, Earthjustice, Colorlines, and Feministing. For a full list see here.
Union Hotels - Using the Union Hotel Guide put out by Hotel Workers Rising this feature will show you which hotels have union workers and which are involved involved in a labor dispute.
Information on green businesses as listed in The Green Pages.
Data on Rush Limbaugh advertisers from stoprush.net
